# CLIリファレンス

archlint CLIは、ツールと対話するための主要な方法です。

## 一般的な使用法

```bash
archlint [command] [options]
```

## コマンド

| コマンド                       | 説明                                                           |
| ------------------------------ | -------------------------------------------------------------- |
| [`scan`](/ja/cli/scan)         | 1回限りのアーキテクチャ分析を実行します                        |
| [`diff`](/ja/cli/diff)         | 現在の状態をベースラインと比較します                           |
| [`snapshot`](/ja/cli/snapshot) | 現在の状態をJSONファイルに保存します                           |
| [`watch`](/ja/cli/watch)       | リアルタイムのフィードバックのためにウォッチモードで実行します |

## グローバルオプション

| オプション            | 説明                                       |
| --------------------- | ------------------------------------------ |
| `-c, --config <path>` | 設定ファイルへのパス                       |
| `-v, --verbose`       | 詳細なログを有効にします                   |
| `-q, --quiet`         | CIフレンドリーモード（プログレスバーなし） |
| `-V, --version`       | バージョン情報を表示します                 |
| `-h, --help`          | コマンドのヘルプを表示します               |
