# archlint watch

`watch`コマンドは、バックグラウンドでarchlintを実行し、ファイルが変更されるたびにプロジェクトを再分析します。

## 使用法

```bash
archlint watch [options]
```

## オプション

| オプション        | デフォルト | 説明                                     |
| ----------------- | ---------- | ---------------------------------------- |
| `--debounce <ms>` | `300`      | 再実行する前に、さらなる変更を待ちます   |
| `--clear`         | `false`    | 各実行の前にターミナル画面をクリアします |

## 使用例

### 開発中のリアルタイムフィードバック

```bash
archlint watch --clear --debounce 500
```
