# 巨大なファイル

**ID:** `large_file` | **重要度:** Medium (default)

独自の郵便番号が必要なほど巨大になったファイルを特定します。

## なぜこれが「不吉な臭い」なのか

極端に大きなファイルは、ナビゲートの悪夢です。コードを書くより、スクロールしてシンボルを探す時間の方が長くなります。通常、2000行のファイルは、3〜4つの小さな論理的モジュールがトレンチコートを着て1つのふりをしているだけです。単一責任の原則に違反し、マージコンフリクトをほぼ確実に引き起こします。

## 修正方法

ファイルをより小さく、焦点の絞られたモジュールに分割します。

## 設定

```yaml
rules:
  large_file:
    max_lines: 1000
```
