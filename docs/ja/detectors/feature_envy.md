# 属性の執着

**ID:** `feature_envy` | **重要度:** Medium (default)

属性の執着は、あなたの家の中で何が起こっているかをあなた以上に知っている詮索好きな隣人のようなものです。メソッドが自分自身のデータよりも別のクラスのデータに遥かに関心を持っているように見える場合に発生します。

## なぜこれが「不吉な臭い」なのか

これは誤った場所にロジックがあることを示す典型的なサインです。メソッドが常に別のオブジェクトに手を伸ばしてデータを引き出し、計算を行っている場合、そのロジックはおそらく別のオブジェクトの中に属しています。カプセル化を破壊し、クラスを密結合にします。

## 修正方法

そのメソッド（または執着があるメソッドの部分）を、使用しているデータを持つクラスに移動します。
