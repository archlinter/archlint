# テストの漏洩 (Test Leakage)

**ID:** `test_leakage` | **重要度:** High (default)

テストファイルやモックファイルからインポートしているプロダクションコードを特定します。

## なぜこれが「不吉な臭い」なのか

プロダクションコードがテストコードに依存すべきではありません。これは、バンドルサイズの増加、セキュリティリスクの原因となる可能性があり、テストファイルがプロダクションビルドから除外されている場合にビルドが失敗する原因となります。

## 修正方法

- 共有ロジックをテストファイルから、プロダクションで安全に使用できる場所に移動します。
- インポートパスが正しいことを確認してください。
