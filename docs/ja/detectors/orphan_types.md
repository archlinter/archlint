# 孤立した型

**ID:** `orphan_types` | **重要度:** Low (default)

定義されているが、変数、引数、または戻り値の型として一度も使用されていない型やインターフェースを特定します。

## なぜこれが「不吉な臭い」なのか

デッドコードと同様に、孤立した型は、何の利益ももたらさずにプロジェクトを散らかし、開発者の認知負荷を増大させます。

## 修正方法

未使用の型を削除してください。
