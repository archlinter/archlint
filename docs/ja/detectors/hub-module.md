# ハブモジュール

**ID:** `hub_module` | **Severity:** Medium (default)

「ハブモジュール」とは、依存関係グラフの中で中心点として機能し、高いファンインと高いファンアウトの両方を持つモジュールのことです。

## なぜこれがコードの不吉な臭い（smell）なのか

ハブモジュールは、アーキテクチャにおける危険な「単一障害点」です。非常に多くのものがこれらに依存し、またこれら自身も多くのものに依存しているため、非常に脆弱であり、コードベース全体に波及効果をもたらさずに変更することが極めて困難です。

## 修正方法

ハブを解消しましょう！ハブを通過するデータや制御のさまざまなパスを特定し、それらを別々の、より焦点の絞られたモジュールに抽出します。
