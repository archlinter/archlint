# 散弾銃の手術 (Shotgun Surgery)

**ID:** `shotgun_surgery` | **Severity:** Medium (default)

散弾銃の手術は、要件の単一の変更によって、多くの異なるモジュールに対して多数の小さな変更を行う必要がある場合に発生します。

## なぜこれが不吉な臭い（smell）なのか

これは、責務がコードベース全体に薄く分散しすぎていることを示しています。変更が困難になり、時間がかかり、エラーが発生しやすくなります。

## 修正方法

- **メソッド/フィールドの移動 (Move Method/Field)**: 関連するロジックを単一のモジュールに集約します。
- **クラスのインライン化 (Inline Class)**: あるクラスが、常に別のクラスと一緒に使用されるメソッドの集まりにすぎない場合は、それらを結合します。
