# Поддержка React

Компоненты React имеют иные архитектурные характеристики, чем традиционные классы или модули.

## Основные возможности

- **Распознавание компонентов**: Идентифицирует компоненты React по паттернам именования и использованию JSX.
- **LCOM отключен**: Автоматически отключает детектор низкой связности (LCOM) для компонентов, так как они по своей сути сосредоточены на состоянии UI и рендеринге.
- **Анализ хуков**: Понимает, что пользовательские хуки являются точками входа для общей логики UI.

## Рекомендуемая конфигурация

```yaml
framework: react
```
