# Поддержка Next.js

Проекты Next.js имеют уникальную маршрутизацию на основе файлов и паттерны сборки, которые archlint понимает.

## Основные возможности

- **Знание маршрутизации**: Автоматически распознает файлы в директориях `pages/` и `app/` как точки входа.
- **Barrel-файлы**: Смягчает правила для barrel-файлов для распространенных паттернов Next.js.
- **Клиентские/Серверные компоненты**: (Скоро) Специализированный анализ для утечки кода между сервером и клиентом.

## Рекомендуемая конфигурация

```yaml
framework: nextjs

entry_points:
  - 'src/pages/**/*.tsx'
  - 'src/app/**/*.tsx'
```
