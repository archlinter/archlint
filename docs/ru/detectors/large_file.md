# Большой файл

**ID:** `large_file` | **Степень критичности:** Средняя (по умолчанию)

Идентифицирует файлы, которые разрослись настолько, что им, кажется, пора выдавать собственный почтовый индекс.

## Почему это «запах»

В огромных файлах невозможно ориентироваться. Вы тратите больше времени на скроллинг и поиск символов, чем на само написание кода. Обычно файл на 2000 строк — это просто три-четыре нормальных модуля, натянутых на один глобус. Это нарушает принцип единственной ответственности и гарантирует конфликты при каждом мерже.

## Как исправить

Разбейте файл на более мелкие, сфокусированные модули.

## Конфигурация

```yaml
rules:
  large_file:
    severity: medium
    max_lines: 1000
```

### Параметры по умолчанию

- `lines`: `1000` - максимальное количество строк в файле
