# Обзор детекторов

`archlint` поставляется с более чем 28 встроенными детекторами, разделенными по типам архитектурных проблем или проблем качества кода, которые они выявляют.

## Проблемы зависимостей

| Детектор                                         | ID                   | Описание                                   | По умолчанию |
| ------------------------------------------------ | -------------------- | ------------------------------------------ | ------------ |
| [Циклические зависимости](/ru/detectors/cycles)  | `cycles`             | Циклические зависимости между файлами      | ✅           |
| [Циклы типов](/ru/detectors/circular-type-deps)  | `circular_type_deps` | Циклы зависимостей только на уровне типов  | ❌           |
| [Циклы пакетов](/ru/detectors/package-cycle)     | `package_cycles`     | Циклы зависимостей между пакетами          | ❌           |
| [Нарушение слоев](/ru/detectors/layer-violation) | `layer_violation`    | Нарушения определенных архитектурных слоев | ❌           |
| [Нарушение SDP](/ru/detectors/sdp-violation)     | `sdp_violation`      | Нарушения принципа устойчивых зависимостей | ❌           |
