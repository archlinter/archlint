# Обзор детекторов

`archlint` поставляется с более чем 28 встроенными детекторами, разделенными по типам архитектурных проблем или проблем качества кода, которые они выявляют.

::: tip
**Ложноположительные срабатывания**: Архитектурный анализ может иногда приводить к ложноположительным результатам, особенно в проектах с интенсивной динамической загрузкой, рефлексией или сложными DI-контейнерами.
:::
