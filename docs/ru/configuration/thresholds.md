# Пороги

Пороги (thresholds) позволяют точно настроить, когда детектор должен сообщать о запахе.

## Общие пороги

| Детектор       | Опция              | По умолчанию | Описание                                           |
| -------------- | ------------------ | ------------ | -------------------------------------------------- |
| `cycles`       | `exclude_patterns` | `[]`         | Glob-паттерны для игнорирования при поиске циклов  |
| `god_module`   | `fan_in`           | `10`         | Макс. входящих зависимостей                        |
| `god_module`   | `fan_out`          | `10`         | Макс. исходящих зависимостей                       |
| `god_module`   | `churn`            | `20`         | Макс. коммитов git в истории                       |
| `god_module`   | `max_lines`        | `500`        | Макс. строк кода в файле                           |
| `complexity`   | `max_complexity`   | `15`         | Макс. цикломатическая сложность на функцию         |
| `deep_nesting` | `max_depth`        | `4`          | Макс. глубина вложенности блоков                   |
| `long_params`  | `max_params`       | `5`          | Макс. параметров на функцию                        |
| `large_file`   | `max_lines`        | `1000`       | Макс. строк на файл                                |
| `lcom`         | `threshold`        | `1`          | Макс. разрешенных несвязанных компонентов в классе |

## Пример конфигурации

```yaml
thresholds:
  god_module:
    fan_in: 20
    max_lines: 800

  complexity:
    max_complexity: 10

  large_file:
    max_lines: 2000
```
