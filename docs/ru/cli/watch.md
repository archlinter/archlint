# archlint watch

Команда `watch` запускает archlint в фоновом режиме и повторно анализирует ваш проект при каждом изменении файла.

## Использование

```bash
archlint watch [options]
```

## Опции

| Опция             | По умолчанию | Описание                                      |
| ----------------- | ------------ | --------------------------------------------- |
| `--debounce <ms>` | `300`        | Ожидание новых изменений перед перезапуском   |
| `--clear`         | `false`      | Очищать экран терминала перед каждым запуском |

## Примеры

### Живая обратная связь во время разработки

```bash
archlint watch --clear --debounce 500
```
