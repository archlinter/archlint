# archlint snapshot

Команда `snapshot` фиксирует текущее состояние архитектуры вашего проекта и сохраняет его в JSON-файл. Этот файл затем можно использовать с командой `diff`.

## Использование

```bash
archlint snapshot [options]
```

## Опции

| Опция                 | По умолчанию              | Описание                      |
| --------------------- | ------------------------- | ----------------------------- |
| `-o, --output <file>` | `.archlint-snapshot.json` | Файл для сохранения snapshot  |
| `-p, --path <path>`   | `.`                       | Путь к анализируемому проекту |

## Примеры

### Создание базового состояния для проекта

```bash
archlint snapshot -o .archlint-baseline.json
```
