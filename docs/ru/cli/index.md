# Справочник CLI

CLI archlint — это основной способ взаимодействия с инструментом.

## Общее использование

```bash
archlint [command] [options]
```

## Команды

| Команда                        | Описание                                          |
| ------------------------------ | ------------------------------------------------- |
| [`init`](/ru/cli/init)         | Создать новый файл конфигурации                   |
| [`scan`](/ru/cli/scan)         | Запустить разовый архитектурный анализ            |
| [`diff`](/ru/cli/diff)         | Сравнить текущее состояние с базовым              |
| [`snapshot`](/ru/cli/snapshot) | Сохранить текущее состояние в JSON-файл           |
| [`watch`](/ru/cli/watch)       | Запустить в режиме watch для живой обратной связи |

## Глобальные опции

| Опция                 | Описание                                 |
| --------------------- | ---------------------------------------- |
| `-c, --config <path>` | Путь к конфигурационному файлу           |
| `-v, --verbose`       | Включить подробное логирование           |
| `-q, --quiet`         | Режим для CI (без индикаторов прогресса) |
| `-V, --version`       | Показать информацию о версии             |
| `-h, --help`          | Показать справку по команде              |
