# 分散配置

**ID:** `scattered_config` | **严重程度:** 低 (默认)

识别分散在许多不同文件中而不是集中管理的配置（如环境变量访问）。

## 为什么这是一种坏味道

集中管理配置可以更轻松地：

- 在一个地方查看所有配置选项。
- 提供默认值。
- 在启动时校验配置。
- 更改配置源（例如，从环境变量改为文件或秘密管理器）。

## 如何修复

创建一个中心 `config` 模块，读取并校验所有环境变量，并将它们作为类型化对象导出。
